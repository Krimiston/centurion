constexpr auto cen::is_user_event ( const event_type  type) -> bool
constexprnoexcept

Indicates whether an event type is a user event.

This function considers any event type enumerator in the range [user, last_event) to be a user event.

Parameters
typethe event type to check.
Returns
true if the event type is reserved for user events; false otherwise.
