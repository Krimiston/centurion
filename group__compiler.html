Detailed Description

Provides constexpr utilities for querying the current compiler.

Note
There is no guarantee that the compiler checks are mutually exclusive.

◆ is_debug_build()

constexpr auto cen::is_debug_build ( ) -> bool
constexprnoexcept

Indicates whether or not a "debug" build mode is active.

Note
This is intended to be use with if constexpr-statements instead of raw #ifdef conditional compilation, since the use of if constexpr prevents any branch to be ill-formed, which avoids code rot.
Returns
true if a debug build mode is currently active; false otherwise.
Since
5.3.0
