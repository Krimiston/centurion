Detailed Description

template<typename T>
class cen::basic_rect< T >

A simple rectangle implementation, based on either SDL_Rect or SDL_FRect.

This type is serializable, see the associated serialize function.

Template Parameters
Tthe representation type. Must be convertible to either int or float.
