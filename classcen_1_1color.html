Detailed Description

An 8-bit accuracy RGBA color.

This type is serializable, see the associated serialize function.

This class is designed to interact with the SDL colors, i.e. SDL_Color and SDL_MessageBoxColor. For convenience, there are approximately 140 color constants provided in the colors namespace,

3.0.0
