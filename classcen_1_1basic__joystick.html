Detailed Description

template<typename T>
class cen::basic_joystick< T >

Represents a joystick device.

This class is a part of the owner/handle API, the owner/handle versions of this class are joystick/joystick_handle

The game controller API is built on top of the joystick API, which means that the game controller is higher-level and easier to use.

Since
4.2.0
See also
joystick
joystick_handle

inlinestaticnoexcept

Specifies whether or not joystick event polling is enabled.

If joystick event polling is disabled, then you must manually call basic_joystick::update() in order to update the joystick state.

Note
It's recommended to leave joystick event polling enabled.
Warning
Calling this function might cause all events currently in the event queue to be deleted.
Parameters
