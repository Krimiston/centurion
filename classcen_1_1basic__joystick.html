Detailed Description

template<typename T>
class cen::basic_joystick< T >

Represents a joystick device.

The joystick is lower level than the game controller API, which is built on top of the joystick API. As a result, the game controller API is easier to use and should be preferred over the joystick API.

This class is a part of the owner/handle API, the owner/handle versions of this class are joystick/joystick_handle

◆ query_axis()

auto query_axis ( const int  axis) const -> int16
inlinenoexcept

Returns the position of an axis.

Most modern joysticks let the X-axis be represented by 0 and the Y-axis by 1. It might be a good idea to combat jitter by imposing thresholds on the returned values.

Note
Some joysticks use axes 2 and 3 for extra buttons.
Parameters
axisthe index of the axis to query.
Returns
the axis position.
