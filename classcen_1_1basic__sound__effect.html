Detailed Description

template<typename T>
class cen::basic_sound_effect< T >

Represents a sound effect.

Unlike with the music API, multiple sound effects can be played at the same time, which is the main difference between the music and sound effect APIs.

The supported file formats are:

  • WAVE/RIFF (.wav)
  • AIFF (.aiff)
  • VOC (.voc)
  • OGG (.ogg)
  • VOC (.voc)
