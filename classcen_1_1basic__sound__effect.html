Detailed Description

template<typename T>
class cen::basic_sound_effect< T >

Represents a sound effect.

This class is a part of the owner/handle API, the owner/handle versions of this class are sound_effect/sound_effect_handle

Unlike with the music API, multiple sound effects can be played at the same time, which is the main difference between the music and sound_effect APIs.

The supported file formats are the following:

  • WAVE/RIFF (.wav)
  • AIFF (.aiff)
  • VOC (.voc)
  • OGG (.ogg)
  • VOC (.voc)
