Detailed Description

template<typename T>
class cen::basic_window< T >

Represents an operating-system window.

This class is a part of the owner/handle API, the owner/handle versions of this class are window/handle

◆ window_flags

enum window_flags : uint32

Represents different window features and options.

Values of this enum are intended to be used to create flag bitmasks, that can be used when creating windows and to obtain information from created windows.
