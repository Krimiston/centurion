Indicates whether the currently stored event is of a particular type.

Note, not all event types have dedicated event classes, so you might want to use the overload taking a event type as a parameter to perform more fine-grained type checks.

inlinenoexcept

Indicates whether there is no internal event instance.

There may still be information about the event type, this function just checks whether there is additional information about the event, i.e. a dedicated event type.

Returns
true if there is no internal event; false otherwise.

◆ get() [1/2]

auto get ( ) -> T&
inline

Returns the current event representation.

Use this function if you're certain about the current event type.

An exception is thrown if there is a mismatch between the requested type and the actual current event type. Use the try_get function for a non-throwing alternative to this function.

Template Parameters
Tthe type of the event to obtain.
Returns
a reference to the internal event type.
